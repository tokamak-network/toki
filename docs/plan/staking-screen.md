1. Dashboard 안에 있는 Staking 관련 컴포넌트를 새 페이지 /staking으로 이동
2. staking 페이지는 인터랙티브와 오른쪽에 작용가능한 컴포넌트 조합인 레이아웃을 유지
3. 단순 라인 위주의 UI보다는 실제 오브젝트 느낌의 레이아웃을 스크린과 배경 내에 만들고 그 안에 오퍼레이터 카드나, 스테이크 어마운트, 스테이크 버튼 등이 들어가게 ( 이 부분이 각 항목을 설명하고 선택하는 과정이 순차적으로 이어지게 스토리라인으로 이어지게)
4. 여기에 사용되는 배경 이미지나 에셋, 그리고 추가적으로 필요한 toki 이미지가 있다면 그 에셋들을 생성하기 위한 프롬프트도 같이 생성
5. 스테이킹이 완료되면 업적 시스템에 따라 우리가 만들었던 카드를 하나 해금해주고, 그 카드를 확인하기 위해 자연스럽게 대쉬보드 페이지로 이어지게
